# 07. 25 개인 복습

### 개인적으로 중요함

```shell
- git init > git add . > git commit -m 'message' > git remote add origin <remote url> > git push origin master

- 위 연동 과정, 이미 만들어진 repository의 경우 주소는 <> Code 를 눌러서 복사할 수 있음.

```


```shell
git clone           < 특정 git repository를 가져옵니다.

git clone <url>     < 명령어를 통해 타인의 오픈소스를 복제할 수 있음.

git branch          < 특정 git repository를 가져옵니다.

--------------------------------------------

전체적인 Git 사용 순서
git clone or git init
git branch feature/'name'
git checkout feature/'name'

git add .
git commit -m "message"
git push origin feature/'name'

compare pull&request
merge

```
