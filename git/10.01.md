# 10.01 강의

## [ProDS] 머신러닝 이론 및 데이터 처리

```shell

요약 변수
- 수집된 정보를 분석의 목적에 맞게 종합한 변수
- 단어 빈도, 상품별 구매 금액, 구매량, 영화 매출

파생변수
- 특정한 의미를 갖는 작위적 정의에 의한 변수
- 매우 주관적, 논리적 타당성을 갖춰야 함.
- 구매상품 다양성 변수, 가격 선호대, 라이프 스타일 


데이터 정제
1. 결측값의 이해
- 기록누락, 미응답, 수집오류 등의 이유로 발생
- 결측값이 포함된 자료라도 의미있음, 정보 손실 최소화하도록 결측을 처리해야함.


2. 결측값 처리법
- 완전제거법 : 하나 이상 포함된 자료 제거 / 정보의 손실로 왜곡 될 수 있음

- 평균대체법 : 나머지값의 평균으로 대체 / 추정량의 표준오차가 과소추정되는 문제 있음

- 핫덱대체법 : 동일한 데이터 내에서 결측값이 발생한 관찰치와 유사한 특성을 가진 다른 관찰치의 정보를 이용하여 대체

- 그밖의 처리법 : Regression imputation, kNN imputation 

```
